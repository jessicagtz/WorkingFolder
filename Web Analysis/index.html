Summary


This project analyses the relationship between weather changes and the distance from the equator. A dataset on over 500 cities was created from data from the OpenWeatherMap API.


After creating the dataset, we used Matplotlib to plot various aspects of the weather vs latitude. We took into account temperature, cloudiness, wind speed, and humitidy. This site provides the source data and visualizations created as part of the analysis. Explanations and descriptions of any trends and correlations observed are also included
